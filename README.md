# spider-nodejs
> a simple nodejs web scraper example by scraping 豆瓣电影

### step1 
__install libs__
``` js
    npm install
```
### step2 
__request pages and store into mongodb__
``` js
    node index.js
```
### step3 
__write datas into .json file from mongodb__
``` js
    node fs.js
```


> ps: code and datas are just for studying and learning
